# Smart Playlist Manager (SPM) Server

## Description
The goal of the Smart Playlist Manager (SPM) project is to deliver a mobile
application that provides each user with a personalized and adaptive music experience.
The app leverages a simple algorithm to recommend songs or generate playlists based on
the listenerâ€™s current selected mood, listening history, and interactions with played songs.

## Installation and Execution
1. Create a DigitalOcean app using this GitHub repository.
2. Create a DigitalOcean MongoDB database and add it to the app.
3. Set the appropriate environment variables found in the repository to the values.
4. Build and launch the DigitalOcean app.

## Features
1. Ready-to-go app using a GitHub repository.

## Troubleshooting
1. Ensure all environment variables are correctly set.
2. Ensure ip-address is whitelisted if testing database on local computer. This can be configured
in the app settings on DigitalOcean.
